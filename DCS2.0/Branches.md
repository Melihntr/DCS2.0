Step1-a ve Step1-b Branch Kullanım Açıklaması

-   Step1-a branch: Core Java örneklerinin geliştirilmesi ve test edilmesi için kullanılmaktadır. Bu dal, temel Java konularını ve best practice'leri öne çıkarmaktadır.
    
-   Step1-b branch: Multi-module Maven projelerinin çalışma prensiplerine dair örneklerin yer aldığı daldır. Bu dal, Maven'in modül tabanlı yapısını ve proje yapılandırmasını göstermektedir.
